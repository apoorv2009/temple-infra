5. Current Service Ownership Summary

  • temple-identity-service: app-level registration, sign-in, password verification, sessions.
  • temple-registration-service: temple subscriptions, Shantidhara bookings, donation orders, payment transactions.
  • temple-admin-service: temple onboarding from backend, admins, trustees, temple content, slot availability, approvals.
  • temple-api-gateway: stable mobile API contract and routing to downstream services.
